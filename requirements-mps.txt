# MPS / Apple Silicon dependencies (development)
# Install with: pip install -r requirements-mps.txt
# These are the minimal extra packages needed when developing on macOS with MPS.
# PyTorch MPS support is built-in; no separate GPU package is required.
#
# torch >= 2.4.0 already includes MPS backend — ensure you install the
# standard PyTorch wheel (not the CUDA variant):
#     pip install torch>=2.4.0 torchvision>=0.19.0

# No MPS-specific Python packages are needed beyond the shared requirements.
# This file exists as a placeholder so install scripts can uniformly run:
#     pip install -r requirements.txt -r requirements-mps.txt
# on Apple Silicon without erroring on missing CUDA dependencies.
